Basic Instruction for running the application: 
***********************
# At the command line make sure you in the Version1 folder
Android
1)ionic add platform android @command line
2)ionic build android  @command line
3)ionic run android  @command line


IOS
1)ionic add platform ios @command line
2)ionic build ios  @command line
3)ionic run ios -device @command line


Splash and icon
1) Add a image named splash.png or icon.png depending on which purpose into the resources folder
2) ionic run resources @command line

AppName
1) Edit the <name> </name> in the config.xml to change the app name
